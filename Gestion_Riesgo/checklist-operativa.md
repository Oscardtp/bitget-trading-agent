# Checklist Pre-Operativa Unificada

**Uso:** Evaluar CADA trade antes de ejecutar. Si no se cumple el mínimo → NO operar.

---

## Checklist General (Aplica a toda estrategia)

| # | Criterio | ✅/❌ |
|---|---|---|
| 1 | La estructura del mercado es clara (tendencia o rango identificado) | |
| 2 | Zona de liquidez/OB/FVG identificada | |
| 3 | Señal de entrada confirmada (vela, ruptura, rechazo) | |
| 4 | Volumen acompaña la señal | |
| 5 | SL basado en estructura real (no porcentual fijo) | |
| 6 | TP definido con R:R ≥ 2:1 | |
| 7 | No hay noticias importantes en próximas 2h | |
| 8 | Position sizing calculado correctamente | |

> **Mínimo para operar: 6 de 8 checks ✅**

---

## Checklist por Tipo de Mercado

### Tendencia
- [ ] Estructura HH/HL o LH/LL confirmada
- [ ] EMAs/ADX confirman dirección
- [ ] Entrada en retroceso a zona de valor

### Rango
- [ ] Soporte y resistencia claros
- [ ] Volumen bajo (no hay presión direccional)
- [ ] Rechazo en bandas del rango

### Ruptura
- [ ] Nivel clave identificado
- [ ] Volumen ≥ +20% sobre promedio en ruptura
- [ ] Vela con cuerpo > 60% del rango

---

## Checklist de Gestión de Riesgo

| # | Criterio | ✅/❌ |
|---|---|---|
| 1 | Riesgo por trade: 0.5% – 2% del capital | |
| 2 | SL colocado (detrás de estructura/ATR/pared) | |
| 3 | TP1/TP2 definidos | |
| 4 | Trailing stop planificado | |
| 5 | No estoy en racha de 3 pérdidas seguidas | |
| 6 | No excedí mi riesgo diario/mensual | |

> **Mínimo para operar: 5 de 6 checks ✅**

---

## Checklist Psicológico

| # | Pregunta | ✅/❌ |
|---|---|---|
| 1 | Estoy tranquilo y enfocado (no emocional) | |
| 2 | No estoy "recuperando" pérdidas | |
| 3 | No es un trade por aburrimiento/FOMO | |
| 4 | Sigo mi sistema, no mi intuición | |

> **Si algún check es ❌ → NO operar**

---

## Flujo de Decisión

```
¿Hay Setup Válido?
├─ NO → Esperar
└─ SI → ¿Cumple Checklist General (6/8)?
         ├─ NO → Esperar
         └─ SI → ¿Cumple Gestión de Riesgo (5/6)?
                  ├─ NO → Ajustar o esperar
                  └─ SI → ¿Estoy bien psicológicamente?
                           ├─ NO → Descansar
                           └─ SI → EJECUTAR TRADE
```
